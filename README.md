# Cute Cat Scroller

Bassically you just sit, watch cat pictures and decide whether it's cute or meh.

[Try it out!](https://krecior4.github.io/cute-cat-scroller/)

## Tech stack

* React + TypeScript + Vite + CSS - style and whole app
* Cat as a service (CataaS)- [cataas.com](https://cataas.com/) cat images, the best API ever [buy them a beer](https://buymeacoffee.com/kevinbalicot)
* Eslint - linting
